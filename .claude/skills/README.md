# Installed skills

Project-local copies of the implementation-relevant **superpowers** skills
(`claude-plugins-official/superpowers` v5.1.0), installed at bootstrap so this
handoff repo is self-contained. Each is a `<name>/SKILL.md` discoverable by the
Skill tool.

Execution path for the implementation plan:
`using-superpowers` → `executing-plans` / `subagent-driven-development` →
`test-driven-development` (+ `systematic-debugging`, `requesting-code-review`,
`receiving-code-review`, `verification-before-completion`,
`using-git-worktrees`, `finishing-a-development-branch`,
`dispatching-parallel-agents`, `writing-plans`).

If your environment already has the global superpowers plugin, prefer the
namespaced `superpowers:<name>` versions — they're identical.
